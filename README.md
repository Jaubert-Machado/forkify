<h1 id="app">Forkify App</h1>

# Installing packages

```sh
npm install
```

# Running dev environment

```sh
npm start
```

# Some ingredients and recipes that you can find in the API

- pizza
- pasta
- avocado
- tomato
- grape
- lime
- orange
- salad

_(You can find more search term in this [list](https://forkify-api.herokuapp.com/phrases.html))_
